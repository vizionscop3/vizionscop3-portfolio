# VizionScop3 Portfolio

Professional portfolio website for Lee Aulder (Vizion) showcasing technical projects, creative work, and consultation services. Deployed on Vercel.

## Tech Stack

- **Framework:** Next.js 14.2.5 (App Router)
- **Styling:** Tailwind CSS 3.4.7
- **Animations:** Framer Motion 11.3.8
- **Icons:** Lucide React 0.424.0
- **Analytics:** Vercel Analytics 1.3.1
- **TypeScript:** 5.5.4

## Design System

Material Neubrutomorphism - A fusion of:
- **Material Design** - Elevation, shadows, purposeful motion
- **Neubrutalism** - Bold colors, thick borders, raw typography
- **Neumorphism** - Soft shadows, embossed/debossed effects

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
